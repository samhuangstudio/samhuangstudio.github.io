# AGENTS.md - SamHuang Studio 網站

這是 SamHuang Studio 的個人作品集網站，是一個簡單的靜態 HTML 網站，包含 CSS 和 JavaScript。

## 建置指令

這是一個靜態網站，不需要建置流程。

- **本地預覽**：直接在瀏覽器中開啟 `index.html`，或使用本地伺服器：
  ```bash
  npx serve .
  # 或
  python3 -m http.server 8000
  ```

- **部署**：將變更推送到 `main` 分支，GitHub Pages 會自動部署。

## 測試

沒有正式的議手動驗測試，建證：
- 在 https://validator.w3.org/ 驗證 HTML 有效性
- 測試跨裝置的響應式設計
- 檢查所有連結是否正常運程式碼風作

## 格指南

### HTML
- 使用語義化 HTML5 元素（`<header>`、`<nav>`、`<main>`、`<section>`、`<footer>`）
- `<html>` 標籤包含 `lang` 屬性
- 標籤和屬性使用小寫
- 所有屬性值加上引號
- 正確關閉所有標籤
- 使用 2 個空格縮排

### CSS
- 小型專案可將樣式放在行內或單一外部樣式表
- 對顏色和重複值使用 CSS 變數
- 優先使用 flexbox 和 grid 進行版面配置
- 行動裝置優先的響應式設計
- 使用有意義的類別名稱（kebab-case）

### JavaScript
- 腳本放在 `<body>` 結尾或使用 `defer`
- 使用現代 ES6+ 語法
- 保持函式小而專注
- 預設使用 `const`，需要重新賦值時才用 `let`
- 使用 try-catch 妥善處理非同步操作的錯誤
- 只在複雜邏輯處加上註解

### 無障礙設計
- 圖片包含 `alt` 文字
- 使用正確的標題層級（h1 → h2 → h3）
- 確保足夠的顏色對比度（WCAG AA 最低標準）
- 讓互動元素可透過鍵盤操作

### 效能
- 盡量減少外部依賴
- 使用系統字型或優化網頁字型載入
- 新增圖片前先壓縮
- 對首屏以下的內容使用延遲載入

### 檔案組織
- `index.html` 放在根目錄
- CSS 放在 `/css/` 目錄
- JavaScript 放在 `/js/` 目錄
- 圖片放在 `/images/` 目錄

### Git 工作流程
- 提交訊息使用英文或繁體中文
- 小而專注的提交，搭配描述性的訊息
- 重大變更建立功能分支
- 合併到 main 前先測試

## 開發備註

這是一頁式靜態網站。網站透過 GitHub Pages 從 main 分支自動部署。目前沒有 CI/CD 流程或測試框架。
